# Миг — production notes

Текущий production-этап: v23.

## Закрыто

- Рабочий backend на Railway.
- Реальные посты, истории, видео, места, сообщения и игры без демо-контента.
- Жалобы, блокировки, модерация.
- Удаление аккаунта и публичная страница `/delete-account`.
- Юридические страницы: `/privacy`, `/terms`, `/community-guidelines`, `/moderation-policy`, `/support`.
- Ограничения медиа и сжатие фото.
- Push-уведомления через Expo Push Notifications.

## Push-уведомления

Клиент регистрирует Expo Push Token после входа и отправляет его на `/api/push/register`.
Backend отправляет уведомления при:

- новом сообщении;
- фото/видео в диалоге;
- приглашении в игру;
- принятии/отклонении игры;
- ходе в игре;
- лайке;
- комментарии;
- новой подписке.

Для отключения отправки push на backend можно поставить переменную:

```text
PUSH_DISABLED=true
```
